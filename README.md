# Absolute Zero

## Project info

**URL**: https://absolute-zero.vercel.app

## Working locally

Clone the repository and install dependencies with a recent version of Node.js and npm (the [nvm project](https://github.com/nvm-sh/nvm#installing-and-updating) is recommended for managing versions).

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

Alternatively, edit files directly in GitHub or spin up a GitHub Codespace if you prefer a hosted development environment.

## Technology stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the production bundle with `npm run build`. Deploy the contents of the `dist` directory to your hosting provider of choice (Vercel, Netlify, Cloudflare Pages, etc.). Configure DNS with your provider to point a custom domain at the deployed site if desired.
